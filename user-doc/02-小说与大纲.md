# 小说与大纲

## 故事与角色

“小说管理”用于查看、创建和删除故事。进入“工作台”后，可在“故事构思”和“故事管理”中继续编辑故事卡。

- “故事构思”：根据标题、类型、基调和核心想法生成故事与初始角色草稿。
- “故事管理”：维护故事概要、主线、类型、基调和角色卡。
- 删除故事会影响其关联内容。删除前先确认不再需要该故事下的大纲和稿件。

## 大纲编排

在工作台的“大纲编排”中：

1. 选择故事并创建大纲。
2. 按章节生成场景，或直接编辑章节标题、摘要和场景内容。
3. 需要时为大纲选择已发布世界观。
4. 使用“保存大纲”提交本轮修改。

场景是正文生成和质量分析的最小工作单位。先完成场景目标和摘要，生成正文时会得到更稳定的上下文。

## 进入写作

在“小说创作”中选择故事、大纲和稿件。一个大纲可以保留多份稿件，用于不同版本或写作方向。选择场景后，编辑器会加载该场景的正文。
