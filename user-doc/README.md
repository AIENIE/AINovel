# AINovel 使用手册

本目录面向实际使用 AINovel 的创作者和管理员，不包含部署密钥、内部接口或实现细节。

## 访问入口

- 正式环境：`https://ainovel.aienie.com`
- 本地测试环境：`https://ainovel.localhut.com`
- 创作者通过统一登录进入；管理员使用独立的 `/admin/login` 入口。

## 阅读路线

1. [快速开始](01-快速开始.md)：登录后创建第一部小说或世界观。
2. [小说与大纲](02-小说与大纲.md)：完成故事卡、角色和章节场景。
3. [正文写作](03-正文写作.md)：生成、保存、精雕模式与编辑器协作。
4. [世界观](04-世界观.md)：创建、编辑、预检和生成设定。
5. [素材、知识库与图谱](05-素材-知识库与图谱.md)：管理写作资料与上下文。
6. [质量、版本与导出](06-质量-版本与导出.md)：检查风险、保留版本和导出稿件。
7. [账户与设置](07-账户与设置.md)：积分、提示词、风格和工作台偏好。
8. [管理员手册](08-管理员手册.md)：后台运营、治理和观测边界。
9. [常见问题](09-常见问题.md)：登录、生成、积分与素材检索问题。

## 使用边界

- 生成和润色会消耗项目专属积分；提交前确认当前故事、大纲、稿件和场景选择正确。
- 质量分析用于识别文本和叙事风险，不判断作者是否使用 AI，也不会自动替换正文，除非你在剧情候选修订中明确点击采纳。
- 密码、注册、找回和普通用户会话由统一登录服务管理；AINovel 个人中心不提供本地改密。
