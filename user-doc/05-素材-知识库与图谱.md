# 素材、知识库与图谱

## 素材库

素材库提供四个标签页：

- “素材列表”：按标题或标签筛选当前素材。
- “手动创建”：录入标题、类型、概要、正文和标签。
- “批量导入”：上传 TXT 文件并等待导入任务完成。
- “审核台”：查看待审素材并选择批准入库或驳回。

已审核素材可用于工作台检索和正文生成上下文。向量服务不可用时，系统会保留关键词检索；结果相关性可能下降但基本检索仍可使用。

普通用户页面当前不提供素材合并、删除和引用历史的操作入口。这些治理能力由管理员后台处理。

## 知识库

工作台“知识库”用于维护 Lorebook 条目：

1. 选择故事。
2. 创建、编辑、删除或批量导入条目。
3. 设置条目的分类、优先级、Token 预算和启用状态。
4. 使用实体提取和上下文预览检查条目是否适合当前场景。

## 知识图谱

“知识图谱”按当前故事展示知识库实体和关系。可按关键词查询、创建或删除关系，并在需要时同步图谱。图谱用于辅助理解设定关联，不会自动改写故事正文。
