# 世界观

## 创建与编辑

在“世界构建”中可以查看已有世界、新建草稿或打开编辑器。世界编辑器包含基础信息和模块字段：

- 基础信息：名称、标语、主题、创作意图和备注。
- 模块字段：按后端定义填写世界规则、环境和其他设定。
- 字段精修：选择单个字段并请求 AI 生成改写建议，再决定是否采用。

保存只会保存当前世界，不会自动修改已引用它的故事正文。

## 预检与生成

1. 保存已知设定。
2. 点击“预检/发布”查看缺失字段和待生成模块。
3. 确认发布后，按模块生成缺失内容。
4. 失败的模块会显示失败状态，可在确认配置和积分后重试。

世界观完成后，可在故事构思、大纲或稿件中选择它作为上下文。删除仅对草稿世界开放；已发布或已被使用的世界应先确认影响范围。
