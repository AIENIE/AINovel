# 快速开始

## 登录

1. 打开 AINovel 首页，选择“登录”或“注册”。
2. 页面会跳转到统一登录服务完成认证。
3. 返回 AINovel 后，系统进入 Dashboard。未登录访问受保护页面会自动回到登录入口。

登录成功后，左侧导航提供“工作台”“小说管理”“世界构建”“素材库”和“设置”；账户菜单中可以进入“个人中心”。

## 创建第一部小说

1. 从“小说管理”选择“引导创作”或“传统新建”。
2. 引导创作从一句话想法开始，可逐步选择三个方向，或选择自动完成。
3. 引导完成后，故事、世界观、角色和章节大纲会直接进入工作台。
4. 传统新建仍可填写标题、流派和核心创意后进入原有构思流程。

## 创建第一个世界观

1. 在 Dashboard 或导航中打开“世界构建”。
2. 选择“创建新世界”，填写名称和核心基调。
3. 在世界编辑器中保存基础信息与模块字段。
4. 使用“预检/发布”检查缺失项；确认后按需生成模块内容。

已发布的世界观可在故事、大纲和稿件中作为引用上下文使用。

## 推荐创作顺序

1. 建立故事卡和主要角色。
2. 为故事建立大纲，生成或编辑章节与场景。
3. 新建稿件，选择场景并生成或手写正文。
4. 导入素材、补充知识库和风格画像。
5. 在质量、版本和导出功能中完成检查与交付。
