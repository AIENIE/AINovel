# 账户与设置

## 个人中心与积分

“个人中心”显示账号资料、项目专属积分和通用积分摘要，并支持：

- 输入兑换码领取 AINovel 项目专属积分。
- 将通用积分按当前规则兑换为 AINovel 项目专属积分。
- 查看项目积分流水和兑换历史。

兑换按钮在请求期间会禁用，避免重复提交。密码和普通用户登录由统一登录服务管理，本页不提供本地密码修改。

## 系统设置

“设置”包含五个标签：

- “工作区提示词”：维护故事构思、章节、正文和润色模板，可查看变量帮助并恢复默认。
- “世界观提示词”：维护模块生成、最终整合和字段精修模板。
- “风格画像”：为故事建立、激活和比较风格画像，维护角色声音并生成草案。
- “模型偏好”：查看可用模型、保存个人偏好、查看用量和运行 A/B 对比。
- “工作台体验”：创建布局预设、调整快捷键和设置写作目标。

修改提示词和模型偏好前，先确认它们适用于当前故事。恢复默认会覆盖该类模板的现有内容。
