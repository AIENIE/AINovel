# 模块说明：个人中心与积分

## 入口与权限
- 路由：`/profile`，需登录。
- 展示用户基础信息与资产余额，支持兑换码和通用积分兑换。

## 主要功能
- **积分资产**：显示余额（千分位），提示积分与 Token 兑换关系。
- **兑换码**：输入 AINovel 项目兑换码领取项目专属积分，成功后刷新余额。
- **通用转专属**：调用 pay-service 扣减通用积分，再写入 AINovel 本地项目专属积分账户。
- **流水与订单**：展示本地项目专属积分流水和通用转专属订单快照。
- **安全设置**：密码与登录由统一登录服务管理（本地不提供修改密码）。

## 关键交互
- 兑换与转换请求期间按钮禁用，避免重复提交。
- 成功后调用 `refreshProfile()` 更新全局用户状态。
