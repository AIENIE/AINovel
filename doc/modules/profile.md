# 模块说明：个人中心与积分

## 入口与权限
- 路由：`/profile`，需登录。
- 展示用户基础信息与资产余额，支持签到与兑换。

## 主要功能
- **积分资产**：显示余额（千分位），提示积分与 Token 兑换关系。
- **每日签到**：基于 `lastCheckIn` 判断是否今日已签到；成功后刷新资产。
- **兑换码**：输入兑换码领取资产，成功后刷新余额。
- **安全设置**：密码与登录由统一登录服务管理（本地不提供修改密码）。

## 关键交互
- 签到与兑换请求期间按钮禁用，避免重复提交。
- 成功后调用 `refreshProfile()` 更新全局用户状态。
