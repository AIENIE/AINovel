# 模块说明：后台管理系统

## 入口与权限
- 路由前缀：`/admin`，仅管理员角色可访问。
- 布局：深色主题 AdminLayout，包含侧边导航与统计概览。

## 主要页面
- **仪表盘**：展示用户增长、素材待审、消费指标等概览。
- **用户管理**：查看用户列表、封禁/解封（由 UserService 提供）。
- **积分与兑换码**：创建/查看本地项目兑换码（用于项目专属积分发放）。
- **系统设置**：注册开关、维护模式、签到积分区间等全局配置。

## 关键交互
- 管理操作后刷新列表状态。
- 对危险操作（封禁/解封）进行确认提示。
