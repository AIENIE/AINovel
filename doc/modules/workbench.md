# 模块说明

## 认证
- JWT + Spring Security；默认种子用户 `admin/password`。

## 故事与大纲
- 故事 CRUD、角色管理、简化的“灵感生成”接口。
- 大纲以 JSON 存储章节/场景结构，可整纲保存与占位生成章节。

## 稿件
- 基于大纲的稿件管理，支持场景生成占位、正文保存、自动保存、写作会话心跳与结束上报。
- 三栏可伸缩布局：左侧大纲树（可折叠）、中间编辑器、右侧多功能面板；支持专注模式。
- 左侧大纲树支持多选场景、状态标记、批量导出入口和基础章节/场景管理。
- 中间区域支持多标签场景编辑、未保存标记、命令面板（`Ctrl+K`）和状态栏（字数/会话时长/净增）。
- 右侧面板包含 `copilot/context/version/export/stats/goals` 六个标签页。
- 版本页支持手动检查点、两版本 diff（并排/统一视图）与 AI 变更总结。
- 导出页支持任务创建、模板选择、进度查看与完成后下载链接。
- 设置页的快捷键修改会通过前端事件广播即时生效（无需刷新页面）。

## v2 工作台扩展
- 知识库页：Lorebook 条目管理、批量导入、实体提取审核、上下文窗口预览。
- 知识图谱页：按关键词查询实体节点并查看详情。
- v2 联调页：覆盖上下文/风格/分析/版本/导出/模型/工作台接口的快速联调入口。

## 素材库
- 手工创建、TXT 上传轮询、列表/审核/查重合并/自动提示与检索。

## 世界观
- 世界草稿/发布流程，模块化编辑，定义来自后端，可轮询生成状态。

## 设置与提示词
- 工作区提示词、世界提示词读取/保存/重置及帮助元数据。
